Ventajas de ASP.NET Core

  • Capacidad de combinación con otros lenguajes
  • Soporte continuo por parte de Microsoft
  • Soporte MVC
  • Acceso a funcionalidades del .NET framework
  • Comunicación estable con base datos SQL Server

Desventajas de ASP.NET Core

  • Menor rendimiento en relación al sistema de ficheros que Linux
  • Plataforma de desarrollo exclusiva (IDE Visual Studio)
  • No es multiplataforma
  • Mayor tiempo de desarrollo
  • Escaso soporte comunitario
  • Elevado consumo de recursos
